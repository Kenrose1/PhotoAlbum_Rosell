## ğŸ“ Description of Pages and Categories
This application is designed for **easy navigation** and a **clean, formal presentation**. The structure uses Laravel's templating system for a consistent user experience across all pages.

### Core Page Structure

* **Layout** (`app.blade.php`): Serves as the **main template layout** for my application. It contains the essential HTML structure, **navigation bar**, and **footer**, ensuring consistency across all content pages. All other pages extend this layout.
* **Homepage** (`Homepage.blade.php`): Displays a **welcome message** and acts as the **primary entry point** to the entire photo collection.
* **Category Pages** (`/views/pages`): Dedicated view files that render specific collections of images. Each page displays all images relevant to its specific category.

### Categories Included

The photo collection is divided into the following different categories:

* **Nature**: Features images of diverse natural environments, including **forests, parks, and rainforests**.
* **Beach**: Showcases various **beaches** from around the world.
* **Mountains**: Displays a collection of images of **famous mountains** globally.
* **Animals**: Contains photos of different kinds of **unique and diverse animals**.
* **Cars**: Presents a curated collection of **luxurious and notable automobiles**.
